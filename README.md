•	Fasting: 3.9-5.5 mmol/L.

Below normal range:
•	Fasting: <3.8 mmol/L.
•	Post-meal: <3.8 mmol/L.

Prediabetic condition:
•	Fasting: 5.6-6.9 mmol/L.
•	Post-meal: 7.8-6.4 mmol/L

•	Fasting: 5.6-6.9 mmol/L.
•	Post-meal: 7.8-11 mmol/L.
